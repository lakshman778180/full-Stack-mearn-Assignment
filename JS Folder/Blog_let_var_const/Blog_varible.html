Declaring Variable

  1. .var
    • var is function scope
    • var can be re-declared
    • var can be updated

Example:

        
          var name = "John";
          var name = "Doe";
          name = "Smith";
          console.log(name);
  1. .let
    • let is block scope
    • let can't be re-declared
    • let can be updated

Example:

        
          let name = "John";
          let name = "Doe";
          name = "Smith";
          console.log(name);
  1. .const
    • const is block scope
    • const can't be re-declared
    • const can't be updated

Example:

        
          const name = "John";
          const name = "Doe";
          name = "Smith";
          console.log(name);
