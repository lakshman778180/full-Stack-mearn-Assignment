Color Property:

The Basics of the color Property

The color property sets the foreground color of an element, typically applied to text. You can define the color using various formats:

  1. Named Colors

    "Example" 
    2.   CSS
  
    
  color: red;
  color: blue;

    CSS provides a set of predefined color names, like red, blue, green, and gold.

  2. Hexadecimal Colors

    "Example" 
    3.     CSS
    
    
    color: #ff0000; /* Red */
    color: #00ff00; /* Green */

    Hex codes are written as #RRGGBB or shorthand as #RGB.

  3. RGB Colors

    "Example" 
    4.       CSS
          
    
    color: rgb(255, 0, 0); /* Red */
    color: rgb(0, 255, 0); /* Green */

    The rgb() function specifies colors using their red, green, and blue components.

  4. HSL

    "Example" 
    5.       CSS
          
    
    color: hsl(0, 100%, 50%); /* Red */
    color: hsl(120, 100%, 50%); /* Green */

    hsl() uses hue, saturation, and lightness.

Color Types:

  1. Primary Colors:

    2. Primary colors are the base colors from which other colors are derived. In web design, the commonly used primary colors are:

    • Red
    • Blue
    • Yellow

  2. Secondary Colors:

    3. Secondary colors are created by mixing two primary colors. For example:

    • Purple (Red + Blue)
    • Green (Blue + Yellow)
    • Orange (Red + Yellow)

Background-color

Property Background-color

The background-color property in CSS is used to define the background color of an HTML element. This property is an essential tool for creating visually appealing web pages and highlighting content effectively.

Synatx

"Example"

The basic syntax of the background-color property is:

css 


element {
  background-color: color;
}
